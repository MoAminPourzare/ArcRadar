# Brand assets

These files are used only to identify outbound links to their respective
platforms. They are not ArcRadar project logos.

- `x-black.png`: X Brand Toolkit
  - https://about.x.com/en/who-we-are/brand-toolkit
- `discord.svg`: Discord Brand Assets
  - https://discord.com/branding
- `github.svg`: GitHub Primer Octicons (`mark-github-24.svg`)
  - https://github.com/primer/octicons

Keep the marks recognizable, preserve their aspect ratios, and follow each
brand owner's current usage guidelines.
